# MATLAB2GITHUB Generates Markdown files for specified directory in docs folder
 
# Examples
```matlab
matlab2github
matlab2github(pwd)
matlab2github('this_folder')
matlab2github(pwd, 'outputdir', 'help')
```
 
# Usage Notes
-  Only one level of nesting is supported
